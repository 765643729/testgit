精灵编辑器（Sprite Editor）

有时一个精灵纹理只包含一个图形元素，但是通常可以更方便的将几个相关的图形组合到一个图像中。例如，图像可以包含一个角色的组成部分，比如轮胎独立于车身移动的汽车。Unity通过精灵编辑器使你可以很容易的从复合图像中提取元素。

注意：

确保您要编辑的图形的纹理类型设置为Sprite（2D和UI）。 关于导入和设置精灵的信息，请查阅Sprites。

具有多个元素的精灵纹理需要在检查器（Inpsector）中把Sprite Mode设置为Multiple。 （见图2：精灵导入检查器（Texture Import Inspector））

打开精灵编辑器

打开精灵编辑器:

  1. 从Project View中选择你要编辑的2D图片。 （图1：Project View）。

    注意你无法编辑Scene View中的精灵。

  2. 点击Texture Import Inspector中的Sprite Editor按钮（图二：纹理导入检查器（Texture Import Inspector））以及Sprite Editor界面（图三：Sprite Editor）。

注意：你只有在你选择的图片的Texture Type设置为Sprite (2D and UI)时你才可以看到Sprite Editor按钮。

注意：如果你的图片有多个元素，在Texture Import Inspector中把Sprite Mode设置为Multiple。

在复合图像时，你会在窗口顶部栏位看到一些控件。右上角的滑块控制缩放，左侧的颜色条按钮选择是查看原始图片还是图片的alpha级别。最右边的滑块控制纹理的像素化（mipmap）。把滑块向左移动减少精灵纹理的分辨率。最重要的控件是左上角的Slice菜单，它提供自动分离元素的选项。Apply和Revert按钮可用保存或忽略你所做的更改。

使用编辑器

使用编辑器最直接的方式是手动识别元素。当你点击图片，你会看到一个角落有手柄的矩形的选区。你可以拖动手柄或矩形的边缘来调整指定元素的大小。隔离元素后，您可以通过在图像的单独部分拖动一个新矩形来添加另一个元素。当你选择一个矩形后，窗口右下角会出现一个面板：

通过面板中的控制器可以设置精灵图形的名称，以及通过坐标设置矩形位置和大小。可以设置像素级上、下、左、右的边的宽度。也可以使用Unity的图片坐标原点和主要“锚点”设置精灵的中心点。你可以选择一组默认的矩形相对位置（比如，中心、右上角等）或者使用自定义坐标。

Slice菜单项旁边的Trim按钮可以调整矩形大小，使其可以基于透明度紧密围绕图形边缘。

注意：Borders只支持UI系统，不适用于2D SpriteRenderer.

自动切片（Automatic Slicing）

手动分离精灵矩形效果很好，但是在许多情况下，Unity可以自动检测图形元素并且分离它们。点击控制栏的Slice菜单，你会看到如下界面：

把切片类型设置为Automatic，编辑器将通过透明度猜测精灵元素的边界。你可以为每个识别出的精灵设置默认的中心点。在Method菜单可以在窗口中选择怎样处理已有的选项。Delete existing选项会直接替换掉所选择的，Smart会在保留或调整已存在的矩形时尝试创建新的矩形，Safe会不改变任何已有的矩形的情况下添加新的矩形。

Grid by Cell Size或者Grid by Cell Count选项也适用于切片类型。这对于在创建时就设计好的精灵十分有用：

Pixel Size值决定了小方块（tiles）的高度和宽度的像素大小。如果你选择grid by cell count，Column & Row决定了切片的行和列的数量。你也可以使用Offset值从图片左上角来改变网格的位置 and the Padding values to inset the sprite rectangles slightly from the grid. The Pivot can be set with one of nine preset locations or a Custom Pivot location can be set.

Note that after any of the automatic slicing methods has been used, the generated rectangles can still be edited manually. You can let Unity handle the rough definition of the sprite boundaries and pivots and then do any necessary fine tuning yourself.

Polygon Resizing

Open the Sprite Editor for a polygon and you have the option to change its shape, size, and pivot position.

Shape

Enter the number of sides you want the polygon to have in the Sides field and click Change.

Size and Pivot

SIZE: To change the polygon’s size, click on the sprite to display green border lines and the Sprite information box. Click and drag on the green lines to create the border you want, and the values in the Border fields change. (Note that you cannot edit the Border fields directly.)

PIVOT: To change the polygon’s pivot point (that is the axis point the polygon moves around), click on the image to display the Sprite information box. Click on the Pivot drop down menu and select an option. This displays a blue pivot circle on the polygon; its location depends on the pivot option to you have selected. If you want to change it further, select Custom Pivot and click and drag on the blue pivot circle to position it. (Note that you cannot edit the Pivot fields directly.)
